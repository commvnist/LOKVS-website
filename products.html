Products

Role-agnostic UWB relative positioning for mobile autonomous fleets.

Our first product is a subsystem-level positioning module for moving-node networks that do not require fixed anchors as a permanent assumption.

Availability: Early R&D · Design-partner access

What is different from conventional RTLS framing

Typical RTLS pattern

  • Static infrastructure assumptions dominate initial topology.
  • Node roles are often predetermined and slow to reconfigure.
  • Integration focuses on localization only.

LOKVS pattern

  • Role-agnostic nodes in motion, continuously resolving relative state.
  • Topology changes handled in software and host-level contracts.
  • Designed as an autonomy subsystem input, not a standalone tracker.

Current product attributes

Output model

3D relative position and confidence outputs at real-time cadence for cooperative node planning.

Deployment target

Mobile fleets, small UAV teams, and ground robot groups where infrastructure is constrained.

Integration stance

Built to sit within existing autonomy software stacks through defined interfaces and explicit timing assumptions.

Planned roadmap structure: Relative positioning, spatial sensing, detection modules, and embedded autonomy interfaces.
